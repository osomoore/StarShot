# Deprecated Deck Set

`core_0_2` is legacy StarShot deck data.

Do not use this deck set for new gameplay work, AI battle analysis, balance tests, or new v2 tests. It remains in the repo only for explicit backward-compatibility checks around older rules, serialization, and persistence behavior.

Use the admin-selected active deck set for current gameplay unless a test or user request specifically names `core_0_2`.
